# jobcan 自動化プロジェクト

jobcan 内に構築したフローから送信されるメール内にある
申請 ID から各種データを取得してスプレッドシートへ展開するプロジェクト

## こちらのプロジェクトは名刺データの取得からサンプルの作成に使用しています

### フロー

1. jobcan により申請が発生する
1. jobcan から申請通知メールが配信される
1. 通知されたメールから申請 NO を取得する（スクリプト化可能領域）
1. jobcan スプレッドシートを開いてメニューからジョブカン接続を押下する(スクリプト化可能領域)
1. 表示されたページの申請 NO 打ち込み欄へ打ち込みを実施(スクリプト化可能領域)
1. スプレッドシートへ jobcan 申請データが配信される
1. スプレッドシートに配置している関数で整形
1. サンプルスプレッドシートを作成して完了

### スクリプト化可能領域からしたの自動化

1. メールではなく申請で進行中の申請 NO を API をたたいて取得する
1. LOG シートにある場合は作成完了の為作成しない
1. LOG シートにない場合は名刺作成関数を呼びだし
